# Composite

여러 개의 객체들로 구성된 복합 객체와 단일 객체 구별없이 사용할  수 있도록 한다   &#x20;

* 전체-부분의 형태를 정의할 떄 유용하게 사용
* 사용자는 동일한 인터페이스를 사용할 수 있다.
